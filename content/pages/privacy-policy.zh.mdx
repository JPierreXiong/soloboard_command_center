**生效日期：2026年2月11日**

## 1. 简介

SoloBoard（"我们"、"我们的"或"我们"）致力于保护您的隐私。本政策说明了当您使用我们的统一监控仪表盘平台时，我们如何收集、使用和保护您的信息。

## 2. 我们收集的数据

我们仅收集提供服务所必需的信息：

**身份和联系数据**：电子邮件地址、姓名。用途：账户创建、身份验证和服务通知。

**财务数据**：付款历史和订阅等级（免费/基础/专业）。我们不存储完整的信用卡号码；付款由安全的第三方（Stripe、Creem）处理。

**技术数据**：IP 地址、浏览器类型和登录日志。用途：安全监控和欺诈预防。

**API 凭证**：您的第三方 API 密钥（Google Analytics、Stripe 等）。重要提示：这些凭证在存储前使用 AES-256-CBC 加密。我们仅使用它们代表您获取数据，绝不与第三方共享。

**使用数据**：仪表盘查看、站点配置和同步日志。用途：提供和改进我们的服务。

## 3. 数据安全

您的 API 密钥和敏感数据通过以下方式受到保护：

**加密**：所有 API 密钥在存储前使用 AES-256-CBC 加密。

**安全传输**：所有数据通过 HTTPS 传输。

**访问控制**：只有授权系统可以访问加密凭证，且仅用于同步您的数据。

**不共享数据**：我们绝不出售或与第三方共享您的数据用于营销目的。

## 4. 处理的法律依据（GDPR）

我们基于以下原因处理您的数据：

**合同必要性**：提供仪表盘监控和数据同步服务。

**法律义务**：用于税务记录或监管合规。

**合法利益**：维护我们平台的安全性和完整性。

## 5. Cookie 和第三方技术

我们使用 Cookie 来增强您的体验：

**必要 Cookie**：用于维护您的会话和身份验证。

**分析 Cookie**：我们使用分析来了解用户如何与我们的平台互动。您可以通过浏览器设置选择退出。

## 6. 数据保留

**活跃账户**：只要您的账户处于活跃状态，数据就会被保留。

**非活跃账户**：如果账户超过 12 个月未活动，我们可能会根据数据保留政策删除相关数据。

**API 密钥**：在账户删除或您移除数据源时立即删除。

## 7. 您的权利

根据 GDPR 和 CCPA，您有权：

- 访问您的个人数据
- 纠正不准确的数据
- 请求删除您的数据（"被遗忘权"）
- 请求数据可移植性
- 选择退出营销通信

## 8. 第三方服务

SoloBoard 与第三方服务（Google Analytics、Stripe、Shopify 等）集成。当您连接这些服务时，您授权我们根据其各自的隐私政策访问数据。我们建议您查看他们的政策。

## 9. 本政策的变更

我们可能会不时更新本隐私政策。我们将通过在此页面上发布新政策并更新"生效日期"来通知您任何变更。
