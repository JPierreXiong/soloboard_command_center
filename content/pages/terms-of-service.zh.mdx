## 1. 简介和条款接受

欢迎使用 SoloBoard（可通过 [https://www.soloboard.app](https://www.soloboard.app) 访问）。SoloBoard 为独立创业者提供统一的监控仪表盘，用于跟踪多个网站的分析、收入和在线状态。通过访问我们的服务，您同意受这些条款的约束。如果您不同意，请不要使用我们的服务。

## 2. 服务使用

SoloBoard 提供仪表盘监控和数据聚合服务。您同意根据所有适用法律以及您连接的第三方平台（Google Analytics、Stripe 等）的服务条款使用本服务。

## 3. 用户账户和责任

要访问 SoloBoard 功能，您必须创建一个账户。您负责：

- 维护账户凭证的安全
- 在您账户下发生的所有活动
- 确保您的 API 密钥和凭证有效且已授权
- 在连接集成时遵守第三方服务条款

## 4. 订阅计划和付款

SoloBoard 采用订阅模式（免费、基础、专业）：

**免费计划**：有限功能，48 小时同步间隔
**基础计划**：$19.9/月，增强功能和 6 小时同步
**专业计划**：$39.9/月，高级功能和 60 分钟同步

付款由第三方提供商（Stripe、Creem）安全处理。价格可能会提前 30 天通知后更改。您同意支付与您选择的计划相关的所有费用。

## 5. 退款政策

我们为付费计划提供 14 天退款保证。要申请退款，请在购买后 14 天内联系 support@soloboard.app。14 天后或如果您违反了这些条款，则不提供退款。

## 6. 数据所有权和使用

**您的数据**：您保留对数据的所有权利。我们不声称拥有您通过 SoloBoard 查看的任何数据。

**API 密钥**：您授予我们仅使用您的 API 密钥代表您获取和显示数据的权限。

**聚合数据**：我们可能使用匿名化的聚合数据进行分析和服务改进。

## 7. 禁止活动

您同意不：

- 通过我们的服务存储或传输恶意代码
- 尝试未经授权访问我们的系统
- 逆向工程或复制我们的软件
- 将服务用于任何非法目的
- 与他人共享您的账户
- 超过连接服务的 API 速率限制

我们努力实现 99.9% 的正常运行时间，但不能保证不间断的服务。我们不对以下情况负责：

- 临时服务中断

- 由于外部因素导致的数据同步延迟

我们非常重视数据安全。您的 API 密钥使用 AES-256-CBC 加密。有关我们如何收集、使用和保护您的数据的详细信息，请参阅我们的隐私政策。

在法律允许的最大范围内，SoloBoard 不对以下情况负责：
